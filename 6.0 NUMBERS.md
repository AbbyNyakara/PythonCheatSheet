## 6.1 Integers 

- This is a whole number.
- Can be positive or negative
- Does not contain any decimals
- To check whether the variable is of type integer or to convert it to an integer, use int()

```python 
num_1 = 2
num_2 = -125
num_3 = 1239454854

print(type(num_1))
print(type(num_2))
print(type(num_3))

>> <class 'int'>
>> <class 'int'>
>> <class 'int'>
```

- Converting to integer data type:
```python
float_num = 12.56
print(int(float_num))
```
